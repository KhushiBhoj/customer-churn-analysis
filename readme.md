# Customer Churn Analysis

This project aims to predict customer churn for a telecommunications company and provide actionable insights to support **data-driven retention strategies**. It includes exploratory data analysis, feature engineering, risk segmentation, and predictive modeling with business recommendations designed for both technical and non-technical stakeholders.
